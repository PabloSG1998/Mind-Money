# Borrador del Proyecto: Mind&Money - Gestor Integral de Vida

## Descripción del Proyecto
Mind&Money es una aplicación móvil diseñada para el ecosistema Android que integra, en una sola plataforma, una básica gestión de finanzas personales y la administración de tareas y proyectos. El objetivo principal es ofrecer al usuario una herramienta de productividad que fomente el ahorro y el cumplimiento de metas personales mediante la gestión personal y mensajes motivacionales y consejos.

## Exposición del Problema
En la era tecnológica actual, se supondría que la tecnología y evolución se presta como un compañero para nuestro progreso y crecimiento personal, pero en muchos solo ha servido como un medio de distracción y poca productividad. Además, muchas apps dependen de una conexión constante a internet, lo que limita su uso en zonas de baja conectividad o compromete la privacidad del usuario.

## Plataforma
La aplicación se desarrollará exclusivamente para la plataforma Android. Se utilizará el lenguaje Kotlin, y tal vez Java, y el entorno de desarrollo se usa Android Studio, apuntando a una compatibilidad mínima de Android 8.0 (API 26) para asegurar un alto alcance.

## Interfaz de Usuario e Interfaz de Administrador
*   **Interfaz de Usuario (App):** Diseño limpio y minimalista. Contará con un menú de navegación con secciones para: Dashboard (resumen), Finanzas (ingresos/gastos), Tareas (lista de pendientes) y Mensajes (de Admin y capsula de tiempo).
*   **Interfaz de Administrador:** Se implementará un módulo de recepción de datos simple donde la app consumirá un servicio web para mostrar consejos diarios de inversión o motivación enviados de forma remota, así como mensajes de remisión.

## Funcionalidad
1.  **Gestión Financiera (Offline):** Registro de transacciones con categorías y visualización de saldo total.
2.  **Gestión de Tareas (Offline):** Creación de tareas, proyectos y recordatorios.
3.  **Cápsula del Tiempo (Offline):** Función para programar mensajes que el usuario recibirá futuramente.
4.  **Sincronización de Consejos (Online):** Recepción de tips de educación financiera y frases motivacionales.
